## 懒加载

在开发大型应用时经常需要用到懒加载，缩短首屏时间，提高用户体验；

本文主要介绍以下几种：
- 自定义高阶组件进行按需加载
- 使用react-loadable小工具库
- 使用getComponent(仅限于3.x,[demo](https://github.com/mosbyxsy/react-lazy-loading-getcomponent))


具体实现参考[demo](https://github.com/mosbyxsy/react-lazy-loading)
